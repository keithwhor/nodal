Welcome to Your Nodal Application!


View the controller that renders this template ./app/controllers/index_controller.js to change the data sent to it, or modify the template directly.

We really recommend you start playing with creating RESTful API endpoints using the built in commands:
  • nodal db:create
  • nodal db:prepare
  • nodal g:model --user
  • nodal g:controller v1 --for:user
  • nodal db:migrate
To get a good feeling about how Nodal works and its intended use.

You can see all of the built in nodal commands by running nodal help

Important Note:
For database usage, please make sure you have Postgres installed. You must also uncomment this.useDatabase and the database import in ./app/app.js. Please also check to make sure your database credentials are correct in ./config/db.json - Happy Developing!
